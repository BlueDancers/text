## git常用命令

git add .**添加指定文件到暂存区**

git status  **显示有变更的文件**

git commit -m "first commit" **提交暂存区到仓库区**

git push **提交到当前分支**

git pull **拉取当前分支最新代码**

git status **查看缓存的文件**

git branch **查看分支**

git branch dev **创建dev分支**

git checkout dev **切换到dev分支**

git merge dev **合并dev分支到当前分支**

git branch -d dev **删除dev分支**

git log **显示当前分支的版本历史**
